# Esercizio 6 / Exercise 6

**Italian:** Date le ore lavorative giornaliere [40, 35, 45, 38, 42] e una paga base di 15€/ora, calcola lo stipendio totale settimanale applicando una maggiorazione di 5€ per ogni ora straordinaria oltre le 40 giornaliere.

**English:** Given the daily working hours [40, 35, 45, 38, 42] and a base pay of €15/hour, calculate the total weekly salary by applying a €5 surcharge for each overtime hour beyond 40 daily.
