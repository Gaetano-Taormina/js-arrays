# JS Array

**Italian:**
Questo progetto contiene una serie di esercizi per fare pratica con gli Array in JavaScript.

## Struttura del Progetto

La repository è organizzata in cartelle separate per ciascun esercizio, numerate da `es_1` a `es_30` (incluso un eventuale esercizio extra come `inventory-check`). Ogni cartella contiene le istruzioni specifiche di quell'esercizio e i relativi file sorgente.

## Dettaglio Importante sulle Soluzioni

- **Dall'esercizio 1 all'8**: dovreste evitare di usare i metodi integrati degli array (es. `map`, `filter`, `forEach`) e cercare di risolvere i problemi utilizzando i cicli tradizionali (`for`, `while`).
- **Dall'esercizio 9 al 30**: questi esercizi sono invece pensati per utilizzare e fare pratica con i metodi nativi e avanzati che JavaScript mette a disposizione per gli array.

---

**English:**
This project contains a series of exercises to practice with Arrays in JavaScript.

## Project Structure

The repository is organized into separate folders for each exercise, numbered from `es_1` to `es_30` (including any extra exercise like `inventory-check`). Each folder contains the specific instructions for that exercise and its related source files.

## Important Detail about Solutions

- **From exercise 1 to 8**: you should avoid using built-in array methods (e.g. `map`, `filter`, `forEach`) and try to solve the problems using traditional loops (`for`, `while`).
- **From exercise 9 to 30**: these exercises are instead designed to use and practice the native and advanced array methods that JavaScript provides.
