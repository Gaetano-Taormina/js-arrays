# Esercizio 8 / Exercise 8

**Italian:** Un ospedale ha 7 infermieri con questi turni: ["mattina", "pomeriggio", "notte", "mattina", "pomeriggio", "notte", "mattina"]. Conta quanti infermieri lavorano in ciascun turno.

**English:** A hospital has 7 nurses with these shifts: ["morning", "afternoon", "night", "morning", "afternoon", "night", "morning"]. Count how many nurses work in each shift.
